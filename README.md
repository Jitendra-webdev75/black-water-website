# 🌑 Evocus Black Water – Product Landing Page

A modern animated product landing page built with **React, Tailwind CSS, and GSAP**.
This project showcases a premium water product using smooth scroll animations, responsive layouts, and interactive UI elements.

---

## 🚀 Live Demo

👉 https://black-water-website-tl8y.vercel.app/

---

## 🛠 Tech Stack

- ⚛️ React (Vite)
- 🎨 Tailwind CSS
- 🎞 GSAP + ScrollTrigger
- 🌐 Vercel (Deployment)

---

## ✨ Features

- Smooth scroll-based bottle animation
- Text stagger animations
- Fully responsive layout (desktop + mobile)
- Interactive CTA button animation
- Particle background effect
- Modern gradient UI styling

---

## 📁 Project Structure

```
src/
 ├── assets/        → Images (bottle, splash, stone, grass)
 ├── components/    → Navbar, Particles
 ├── App.jsx        → Main layout + animations
 └── main.jsx       → Entry point
```

## ⚙️ Installation

Clone the repo:

```bash
git clone https://github.com/your-username/your-repo-name.git
```

Go to folder:

```bash
cd your-repo-name
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

---

## 🌍 Deployment

This project is deployed using **Vercel**.

To deploy:

1. Push project to GitHub
2. Import repo in Vercel
3. Click Deploy
